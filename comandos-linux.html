Comandos de Ajuda do Terminal

  • help - Exibe informações de ajuda geral.
  • man comando - Exibe o manual do usuário para um comando específico.
  • --help ou -h - Muitos comandos possuem opções de ajuda curtas.
  • info comando - Exibe informações detalhadas sobre um comando.
  • whatis comando - Exibe uma breve descrição de um comando.

Comandos de Manipulação de Arquivos e Diretórios

Aqui estão alguns comandos úteis para manipular arquivos e diretórios no terminal:

  • ls - Lista os arquivos e diretórios no diretório atual.

    • Variações do Comando ls

    O comando ls é usado para listar arquivos e diretórios. Aqui estão algumas variações:

    • ls -l - Lista os arquivos com detalhes, incluindo permissões, proprietário e tamanho.
    • ls -a - Lista todos os arquivos, incluindo os ocultos que começam com ".".
    • ls -h - Lista arquivos com tamanhos humanamente legíveis (KB, MB, GB).
    • ls -t - Lista arquivos por data e hora de modificação, do mais recente ao mais antigo.
    • ls -R - Lista arquivos recursivamente, mostrando também os diretórios internos.
    • ls -lhS - Combina várias opções. Lista com detalhes, tamanhos legíveis e ordena por tamanho.
    • ls diretório - Lista os arquivos e diretórios no diretório especificado.

  • pwd - Exibe o caminho do diretório atual.

    • Variações do Comando pwd

    O comando pwd é usado para mostrar o diretório atual. Aqui estão algumas variações:

    • pwd - Mostra o caminho absoluto do diretório atual.
    • pwd -P - Mostra o caminho físico real (resolve links simbólicos).
    • pwd -L - Mostra o caminho lógico (não resolve links simbólicos).

  • cd diretório - Navega para o diretório especificado.

    • Variações do Comando cd

    O comando cd é usado para mudar de diretório. Aqui estão algumas variações:

    • cd diretório - Navega para o diretório especificado.
    • cd .. - Navega para o diretório pai (um nível acima).
    • cd ~ - Navega para o diretório do usuário (home).
    • cd - - Retorna para o diretório anterior.
    • cd / - Navega para o diretório raiz.

  • mkdir nome - Cria um novo diretório com o nome especificado.

    • Variações do Comando mkdir

    O comando mkdir é usado para criar diretórios. Aqui estão mais algumas variações:

    • mkdir dir1 dir2 - Cria múltiplos diretórios de uma vez.
    • mkdir -m modo dir - Define permissões específicas para o diretório criado.
    • mkdir --help - Exibe informações de ajuda sobre o comando.
    • mkdir -v dir - Mostra uma mensagem detalhada para cada diretório criado.

  • touch arquivo - Cria um novo arquivo vazio com o nome especificado.

    • Variações do Comando touch

    O comando touch é usado para criar e modificar a data de acesso e modificação de arquivos. Aqui estão algumas variações:

    • touch arquivo - Cria um novo arquivo vazio com o nome especificado.
    • touch -c arquivo - Não cria um arquivo se ele não existir.
    • touch -t YYYYMMDDHHMM arquivo - Define a data e hora do arquivo no formato específico.
    • touch -d "data" arquivo - Define a data e hora de modificação do arquivo.
    • touch -r arquivo_referência arquivo - Define a data e hora do arquivo baseado em outro arquivo.
    • touch -a arquivo - Atualiza somente a data de acesso do arquivo.

  • cp origem destino - Copia um arquivo ou diretório da origem para o destino.

    • Variações do Comando cp

    O comando cp é usado para copiar arquivos e diretórios. Aqui estão algumas variações:

    • cp origem destino - Copia um arquivo ou diretório da origem para o destino.
    • cp -r diretório destino - Copia um diretório e seu conteúdo para o destino.
    • cp -i arquivo destino - Pergunta antes de substituir um arquivo existente.
    • cp -u arquivo destino - Copia apenas se o arquivo de origem for mais novo do que o destino.
    • cp -v arquivo destino - Exibe informações detalhadas durante a cópia.
    • cp -p arquivo destino - Preserva as permissões e datas originais do arquivo.

  • mv origem destino - Move (ou renomeia) um arquivo ou diretório da origem para o destino.

    • Variações do Comando mv

    O comando mv é usado para mover arquivos e diretórios. Aqui estão algumas variações:

    • mv origem destino - Move (ou renomeia) um arquivo ou diretório da origem para o destino.
    • mv -i origem destino - Pergunta antes de substituir um arquivo existente.
    • mv -u arquivo destino - Move apenas se o arquivo de origem for mais novo do que o destino.
    • mv -v arquivo destino - Exibe informações detalhadas durante o movimento.
    • mv -f origem destino - Força a substituição do destino, sem perguntar.
    • mv -u * destino - Move todos os arquivos, mas somente se o destino não existir.

  • rm arquivo - Remove um arquivo. Use -r para remover um diretório.

Lembre-se de que muitos comandos possuem opções adicionais. Você pode usar --help ou -h após um comando para ver mais informações sobre suas opções.
